Create your own scores with HTML5 and JS

The Score Editor is still in pre-aplha phase, due to some issues with multiline rendering that haven't been solved yet... But feel free to try it out! Use the arrow keys to navigate through the score. To add new notes, go to the end and hit Enter. Pitch can only be changed one semitone at a time by now. To change the duration of a single note, hit a number while it's selected. It's the same encoding Finale does: 7 for a whole note, 6 for a half note, 5 for a quarter, and so on.
